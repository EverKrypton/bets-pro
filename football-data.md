## Usage in Bets Pro

1. Go to Admin → Settings → API Keys
2. Paste the API Token in "football-data.org API Key" field
3. Save settings

## Free Tier Limits

- 10 requests per minute
- Covers: Premier League, La Liga, Bundesliga, Serie A, Ligue 1, Champions League, and more

## What it does

Provides real-time live scores for matches in progress:
- Live score updates
- Match minute/countdown
- Score change animations on the sports page
- League badges and team crests
